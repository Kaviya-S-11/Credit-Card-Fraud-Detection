# Credit-Card-Fraud-Detection

• Built a machine learning model to detect fraudulent credit card transactions.
• Used Logistic Regression and handled imbalanced data using undersampling.
• Evaluated model using precision, recall, F1-score, and confusion matrix.
• Tools: Python, Pandas, NumPy, Scikit-learn, Matplotlib.
